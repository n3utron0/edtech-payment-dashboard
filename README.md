# EdTech Payment Analytics Dashboard

## Overview
Interactive Power BI dashboard analyzing 125,000+ payment 
transactions across 4 ERP partners on Edviron's platform.

## Business Context
Three pricing layers govern every transaction — Edviron Buying, Partner Pricing, and Merchant Pricing. This dashboard computes and visualizes revenue distribution across all three layers.

## Dashboard Pages
- Home — Navigation and project overview
- Executive Summary - KPIs, GMV, revenue trend
- Partner Performance - ERP-wise revenue split and take rate
- Gateway & Payment Method - Infrastructure and payment analysis
- Institute Drill Down - School level revenue breakdown

## Key Metrics Tracked
- Total GMV
- ERP Revenue, Edviron Net Revenue, Edviron Gross Revenue
- Partner-wise Average Take Rate
- Institute-level transaction volume

## Tools Used
- Microsoft Excel - Data cleaning and normalization
- Power Query - Data transformation and merging
- Power BI Desktop - Dashboard and DAX measures

## Data
125,000+ transactions | Jan–Aug 2025 | Partners: AKIRA, Yahweh, CampusPro, SparkIT
